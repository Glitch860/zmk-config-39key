# zmk-config-39key
39 key keyboard custom ZMK firmware

Link to ZMK documentation: [https://zmk.dev/docs]

3D printed keyboard link:
[https://www.thingiverse.com/thing:4592203]

link to Single switch PCBs printed with JLCPCB:
[https://github.com/swanmatch/MxLEDBitPCB]

Had to modify around the controller points to fit the nice nano with the battery helper PCB/switch. 

Also had to remove some support pillars to fit the Single switch PCBs for this build which also added underglow RGB.

NiceNano pin out schematic
![image](https://github.com/user-attachments/assets/51da235c-7f70-4ce6-a91d-14201e0008ff)

Used to complete this build:
- 39 MX style switches
- 39 1N4148 diodes
- 39 kailh hot swap sockets
- 39 Single Switch PCBs
- 22 gauge tinned copper wire
- 24 gauge insulated wire
- 1 NiceNano controller
- 1 Battery helper PDB from [https://www.boardsource.xyz/]
- Hot glue if the switches need to be held in (because the pate is 3mm thick, switches need 1.5mm).
